Info aggiuntive per la risoluzione dell'esercizio delle API per le previsioni del tempo

1. per l'API che permette di accedere alle nazioni viene fornito direttamente il file JSON: api-nazioni.json
Il file si può anche ricavare direttamente all'indirizzo https://raw.githubusercontent.com/pmontrasio/codici-stati/master/dist/countries.json

2. l'API che permette di accedere alle previsioni del tempo l'indirizzo è presente all'indirizzo https://openweathermap.org/current

3. La API KEY per accedere alle previsioni del tempo può essere ottenuta registrandosi al servizio. Per sicurezza vi fornisco anche quella che ho utilizzato io: const apikey = "3d8aa45f7271f6bcb67cba7a6b0896d7";
